Scheduled Sales

How Scheduled Sales work: Set a sale price, start time, and end time directly on each product. Products with the same Sale Group Name share one countdown timer on the store — great for a flash sale feel. Products with no group name show their own individual timer. Sales are driven by product data — no extra Firestore reads needed.
